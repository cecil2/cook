# Cook

> 好的，今天我们来做菜 🥬

本项目是一个简单的菜谱搜索工具，旨在帮助用户在食材有限的情况下找到可以制作的菜谱。

## 部署与访问

本项目已部署在 GitHub Pages：[https://cecil2.github.io/cook/](https://cecil2.github.io/cook/)

## 开发

本项目基于 Nuxt 3 开发。

```bash
# 安装依赖
pnpm install

# 转换数据 (CSV to JSON)
pnpm convert

# 启动开发服务器
pnpm dev
```

## 部署

### Vercel

本项目支持一键部署到 Vercel。

### GitHub Pages

本项目配置了 GitHub Actions，推送到 `main` 分支后会自动部署到 GitHub Pages。
